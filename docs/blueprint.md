# **App Name**: BoardPrep Pro

## Core Features:

- Question Fetching: Automatically fetch Class 10 board exam questions (e.g., UP Board) from specified sources (web scraping, APIs).
- Question Tracking: Track attempted questions to avoid repetition for each user using local storage.
- Persistent Storage: Use SQLite for local persistent storage to save questions, user answers, and progress.
- Quiz Categories: Provide at least six quiz categories, each containing a minimum of 20 questions.
- Quiz Attempts: Maintain the scores of the last 10 quiz attempts per user.
- Detailed Feedback: Provide detailed feedback after each quiz, indicating correct/incorrect answers and explanations.
- Adaptive UI: Adaptive layouts with support for all device screen sizes

## Style Guidelines:

- Primary color: A calm blue (#64B5F6) to inspire trust and focus, essential for an educational app.
- Background color: Light grey (#F5F5F5), a light scheme offering a clean, distraction-free backdrop for studying.
- Accent color: A vibrant orange (#FF7043) that provides contrast for key interactive elements, like call-to-action buttons and progress indicators, without causing distraction.
- Body font: 'PT Sans', a humanist sans-serif that balances modernity and readability, ideal for prolonged reading in both questions and feedback.
- Headline font: 'Playfair', a high contrast, elegant, modern serif for headlines. Use 'PT Sans' for body text.
- Use clear, minimalist icons for quiz navigation and feedback.
- Subtle animations for transitions and feedback to enhance the user experience.
